##### 1.快速乘算法

思路：🌰a * b，可将b转化成2进制，在b转化为十进制时，每位除了乘以权重(2*n)外，还要乘以a，即可得到a * b的结果

```java
import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        boolean flag = false;
        int n = in.nextInt();
        int a = 50, b = 50, val = a;
        int ans = 0, mod = (int)(1e9 + 7);
        while(b != 0){ 
            if(b % 2 == 1){ //如果b为奇数，说明b的二进制最低位是1
                ans += val;
            }
            val *= 2;
            b /= 2;  //等价于b的二进制右移操作
            val %= mod;
            ans %= mod;
        }
        System.out.println(ans);
    }
}
```
